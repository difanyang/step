Selected Projects

Calendar

February 2019 | Vanderbilt University

A calendar storing a collection of reminders in the chronological order.

  • Enable users to add, delete, and search for a certain reminder.
  • Examined three different implementations in C++: static arrays, dynamic arrays, and linked lists.
  • Tested the program with 20+ trial cases to ensure its reliability.

Guitar Simulator

March 2019 | Vanderbilt University

A program turning text files into sound files.

  • Designed a program that read a text file with information on what notes were to be generated and when they were to be generated, and from that generated a sound file.
  • Experimented text message storing with both a queue ADT and a stack ADT in C++ and measured the relative performance.

Text Twist

April 2019 | Vanderbilt University

A solution generator for the game Text Twist.

  • Solved the game with the basic principles of trial and error.
  • Constructed a tree-like ADT to store a lexicon of all accepted words.
  • Implemented the main program that searched for all words in the lexicon, of three or more letters, that could be generated from the seven given letters.
